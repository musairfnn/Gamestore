**--ARSITEKTUR--**
1. Arsitektur Sistem
 Secara keseluruhan sistem ini dibuat dengan arsitektur monolit, yakni sistem yang dibuat dengan menggunakan satu server serta satu database dalam satu sistem. Jadi semua layanan seperti catalog, pembayaran dan sebagainya itu berada dalam satu wadah.
2. Arsitektur Website
 Website ini dibuat dengan pola MVC (Model, View, Controller). Model sebagai jembatan komunikasi ke database, View sebagai penampil data yang diambil dari API, Controller sebagai pengendali proses bisnis.
